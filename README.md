# InAmigos Awareness Page

This repository contains a single HTML landing page:

- `Inamigosawareness.html`

## About

The page is a visually rich awareness landing page with sections for navigation, hero, about, projects, impact, gallery, CTA, and footer styling. It uses modern CSS layout features and a warm, nature-inspired palette.

## How to view

Open `Inamigosawareness.html` in a browser, or use a local server if you prefer.

## Notes

- The file includes inline CSS and responsive media queries.
- No additional build tools or dependencies are required.
